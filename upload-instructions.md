# 📤 GitHub Upload Instructions

## 🎯 **MANUEL UPLOAD (Git olmadan)**

### ✅ **ADIM 1: GitHub Repository'yi Aç**
1. https://github.com/HknAkn42/biletpro-complete adresine git
2. "Add file" → "Upload files" butonuna tıkla

### ✅ **ADIM 2: Klasörleri Oluştur**

**Önce klasörleri oluştur:**

#### 📁 **frontend/ Klasörü**
1. "Create new file" deyin
2. File name: `frontend/.gitkeep`
3. Content: boş bırak
4. "Commit new file"

#### 📁 **database/ Klasörü**
1. "Create new file" deyin  
2. File name: `database/.gitkeep`
3. Content: boş bırak
4. "Commit new file"

#### 📁 **docs/ Klasörü**
1. "Create new file" deyin
2. File name: `docs/.gitkeep`
3. Content: boş bırak
4. "Commit new file"

### ✅ **ADIM 3: Dosyaları Upload Et**

**Frontend dosyaları:**
1. `frontend/` klasörüne git
2. "Add file" → "Upload files"
3. Bu dosyaları seç:
   - index.html
   - login.html
   - satis.html
   - musteri.html
   - checkin.html
   - gise.html
   - personel.html
   - rapor.html
   - menu.js
   - theme.css
   - free-backend-starter.js

**Database dosyaları:**
1. `database/` klasörüne git
2. "Add file" → "Upload files"
3. Bu dosyayı seç:
   - database-setup-complete.sql

**Docs dosyaları:**
1. `docs/` klasörüne git
2. "Add file" → "Upload files"
3. Bu dosyaları seç:
   - backend-setup.md
   - deployment-strategy.md
   - supabase-vercel-deployment-guide.md

### ✅ **ADIM 4: README.md Güncelle**

1. Ana sayfada README.md'ye tıkla
2. "Edit this file" deyin
3. Hazırladığım README.md içeriğini yapıştır
4. "Commit changes"

### **Upload Hatası**

- Internet bağlantısı stabil olmalı
- GitHub login kontrol et

### **Klasör Hatası**
- .gitkeep dosyası oluştur
- Klasörler boş kalmamalı

### **Permission Error**

- GitHub login olmalı
